# Acquisition Pulse Website

Modern profitability dashboard landing page for e-commerce businesses combining COGS + LTV:CAC analysis.

## Features

- Clean, modern design with elevated card styling
- How It Works section with step cards and hover effects
- Benefits grid with smooth transitions
- Responsive mobile design (64px logo on mobile, 88px desktop)
- Privacy-first messaging
- Professional comparison table
- Call-to-action sections with gradient backgrounds

## Files

- `index.html` - Landing page with hero, problem, comparison, how-it-works, benefits, CTA sections
- `getting-started.html` - Setup guide for Shopify CSV export and Google Ads connection
- `privacy.html` - GDPR/CCPA-compliant privacy policy
- `styles.css` - Modern styles with card shadows, transitions, and responsive layout
- `assets/` - Logo and favicon files

## Deployment

This website is deployed via GitHub Pages at: https://ajd84.github.io/acquisition-pulse/

To deploy:
1. Copy all files to your repository's `/docs` folder
2. Enable GitHub Pages in repository settings, serving from `/docs` folder
3. Commit and push
4. GitHub automatically rebuilds and deploys

## Design System

- **Colors:** Navy (#1E3A8A), Orange (#FB8C00), Grays (50-900)
- **Typography:** System fonts, 2.25rem headings, 1.6 body line-height
- **Spacing:** 5rem section padding, 2rem card padding, 2rem gaps
- **Shadows:** 0 8px 24px rgba(0,0,0,0.08) on cards, 0 12px 28px on hover
- **Transitions:** 0.2s ease on all interactive elements
- **Breakpoints:** Mobile (max-width: 768px)

## Key Sections

### How It Works
Three-step process with numbered badges, card styling, and hover lift effect.

### Benefits
Six-benefit grid with left-border accent, shadows, and hover transitions.

### Comparison Table
Feature comparison with navy header, zebra striping, and hover states.

All sections include proper ARIA labels and semantic HTML.
